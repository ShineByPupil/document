## 基础类型

```ts
let isFlag: boolean = false; // 布尔值
let num: number = 1; // 数字
let str: string = "string"; // 字符串
let arr1: number[] = [1]; // 数组
let arr2: Array<number> = [1]; // 数组范式

function add(num1: number, num2: number): number {
  return num1 + num2;
}
```

### 元组
