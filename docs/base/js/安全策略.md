# 安全策略 - JavaScript

## 同源策略及限制

> 浏览器的核心安全机制，限制不同源（协议、域名、端口任一不同）的脚本或文档间相互访问资源与数据，防止恶意网站窃取用户隐私或发起攻击。

**约束如下：**

1. DOM 访问：禁止跨域页面通过 JavaScript 操作对方 DOM
2. 数据存储：限制 `Cookie`、`localStorage`、`IndexedDB` 等同源读写，避免敏感信息泄露
3. 网络请求：默认拦截跨域 `Ajax` 请求（遵循 `CORS` 规则前），阻断非授权数据交互
