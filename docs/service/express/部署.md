### 私密变量

- 操作流程
  - Settings -> Secrets and variables -> Actions -> New repository secret
  - 设置 -> 机密和变量 -> 操作 -> 新建仓库机密
- 配置项

| Name 名称         | Secret 机密                           | 说明                  |
| ----------------- | ------------------------------------- | --------------------- |
| `SERVER_IP`       | `101.37.82.81`                        | 服务器ip地址          |
| `SERVER_SSH_KEY`  | `-----BEGIN OPENSSH PRIVATE KEY-----` | SSH密钥，最后保留换行 |
| `SERVER_USERNAME` | `root`                                | 服务器用户名          |
