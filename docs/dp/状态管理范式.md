# 状态管理范式

## 一、有限状态机 (FSM)

```js
class TrafficLight {
    constructor() {
        this.states = ['red', 'yellow', 'green'];
        this.current = 0;
    }

    next() {
        this.current = (this.current + 1) % this.states.length;
    }
}
```

## 二、状态共享模式

```js   
// 使用Proxy实现跨组件状态同步
const globalState = new Proxy({}, {
    set(target, key, value) {
        Reflect.set(target, key, value);
        notifyComponents(key);
        return true;
    }
});
```
